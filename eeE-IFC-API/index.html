The IFC-API

The purpose if the eeE-IFC-API is to provide a general web interface for accessing IFC model server functionality. It is sponsored and developed as a part of the eeEmbedded project, see section below. The interface is presented as a set of web services using REST and JSON as design foundations. The services are documented here

Executive Summary

This web resource contains specification and documentation of web services as defined for the eeEmbedded task 4.4 Multi-Model services. Implementation is provided by the the EDMmodelServer™ from Jotne,one of the BIM Model Servers in the eeEmbedded project.  The server is provided as an server instance available for the duration of the project, and not as a product.

Partners has contributed from their expert viewpoint as follows:

·      EPM:        Responsible for providing necessary hardware, software and supporting components to make the server available.

·      NEM:     Specification of the eeE BIM API implemented on the server

·      IAB:        Specification of the eeE BIM API implemented on the server

·      CIB:        Input to specification and implementation

·      XXX:      contributions to parts xxx

·      ALL Partners: Providing feedback and evaluation
